VS Code的使用

Posted on 2019-12-20 In vscode

主要记录工作中VS Code使用的一些心得和技巧。

Ctrl + P

按下Ctrl + P，VS Code界面出现输入框，可以输入一下内容：

  • 直接输入文件名，可全局搜索文件并打开。如果找某个文件夹下的文件，可以输入文件夹名+/或者\。
  • 输入>，进入VS Code主命令框模式，快捷键是Ctrl + Shift + P。
  • 输入:，跳转当前文件行数。快捷键是Ctrl + G
  • 输入@，跳转symbol(搜索当前文件的变量和函数)，快捷键Ctrl + Shift + O
  • 输入@:，会把跳转的symbol分类。快捷键是Ctrl + Shift + O，然后输入:。
  • 输入?，会列出当前可执行的操作。会列出包括以上输入在内的所有可输入的内容，对应着不同的操作功能。
